In order to run the following code please follow the following steps:
1. CD to the current path of the main.cpp file
2. Open your terminal and input the following command: g++ main.cpp -o main
3. Once the compiler has finished the compilation, a file with either a .exe (Windows) extension or no extension at all (MacOS) will be generated.
4.1. In the Command Prompt, input the following for Windows: main.exe
4.2. In the Terminal, input the following for MacOS: ./main
5. Once the program runs, you may follow the instructions on the screen. You may also use the following commands:
  - exit
  - help
  - nvidia-smi
